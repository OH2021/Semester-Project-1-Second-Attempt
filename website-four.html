Iloveimg.com is a website that's very practical, you can compress, resize, crop, convert to and from jpg, edit photos, watermark images, make memes, rotate images and convert webpages in HTML to jpg or svg. I've actually used it myself for the images found on this website actually and it's super simple and easy to use. You click the "Select images" button like you see in the screenshot, then decide how much you want to resize the filesize and before you download the new resized image you can see how big the file is so that you actually know if it's within the size limit of what you need. Also, I've tried to convert images to jpg which is also very simple and works more or less in the same way with "Select image"-button and so on.
